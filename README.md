# KuittiFlow

KuittiFlow is a privacy-first receipt tracking app designed to help users understand their spending without accounts or cloud services.

All data stays on the user's device. The goal is to provide a simple, fast and transparent way to track expenses using receipts.

---

## Core idea

- Add receipts quickly on mobile
- Track total spending by week and month
- Compare different periods (A vs B)
- No accounts, no cloud sync

---

## Scope (v1)

Included:
- Manual receipt entry (total required, store optional)
- Weekly and monthly summaries
- Period comparison
- Dark mode

Not included (by design):
- OCR
- Product-level categorization
- Cloud sync or user accounts

---

## Technology

- Progressive Web App (PWA)
- Vanilla HTML, CSS and JavaScript
- On-device storage
- No backend
